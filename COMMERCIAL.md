# Commercial licensing

The public codebase is intended to be available under **AGPL-3.0-only**.

Teams that want to use this library in a proprietary product or closed-source hosted service should obtain a separate commercial license from the copyright holder.

A sensible commercial package would usually grant:

- use in closed-source applications
- use in proprietary SaaS deployments
- internal modification without public source release
- optional support / consulting / priority fixes

Keep the trademark separate from the code license.
That lets the ecosystem stay open while the official distribution keeps a clear identity.
